모든 쇼케이스 객체의 최상위 기반 클래스입니다.
이 클래스는 객체의 생명주기와 아이덴티티를 관리하는 기본적인 기능을 제공합니다.

**의도하는 문서 모습:**
- `Showcase/Core` 카테고리 안에 위치합니다.
- `base` 태그가 배지로 표시됩니다.
- 상속 계층의 가장 뿌리가 됩니다.
