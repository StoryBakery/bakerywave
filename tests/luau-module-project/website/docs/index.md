# Bakerywave 테스트 벤치마크

이 사이트는 `bakerywave`의 모든 기능(테마, 문서 생성기, 검색 등)을 검증하기 위한 통합 테스트 벤치마크입니다.

## 🚀 주요 테스트 항목

- **[기능 쇼케이스 (Showcase)](/docs/features-showcase)**: 문서 생성기가 지원하는 모든 태그와 시그니처를 한눈에 확인합니다.
- **[API 레퍼런스](/reference/luau)**: 실제 Luau 소스 코드에서 생성된 클래스 문서를 탐색합니다.
- **[태그 테스트 (Tags Test)](/docs/luau-tags-test)**: 수동으로 작성된 MDX 파일에서의 태그 렌더링을 확인합니다.
- **[링크 테스트 (Links Test)](/docs/luau-links.md)**: 문서 간 연결 및 외부 프로젝트 링크 동작을 확인합니다.

---

이 사이트는 배포용이 아니며, 오직 개발 및 검증 목적으로만 사용됩니다.
모든 문서의 텍스트와 코드는 `bakerywave`가 로블록스 공식 문서 스타일을 얼마나 정확하게 재현하는지 보여주기 위해 정교하게 설계되었습니다.
