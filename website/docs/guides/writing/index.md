# 문서 작성 가이드

환영합니다! 이 가이드는 Bakerywave 프로젝트의 문서를 작성하는 모든 분들을 위한 안내서입니다.
문서 작성 경험이 없더라도 걱정하지 마세요. 아래 가이드를 순서대로 읽으면 누구나 멋진 문서를 만들 수 있습니다.

## 시작하기 전에

Bakerywave 문서는 **가독성**과 **일관성**을 중요하게 생각합니다.
모든 문서는 `docs/` 폴더 아래에 Markdown 파일로 저장됩니다.

## 목차

1. [**Markdown 기본 문법**](./markdown.md)
   - 제목, 목록, 코드 블록 등 가장 기초적인 작성법을 배웁니다.
   - 처음 시작하는 분들은 이 페이지부터 읽어보세요.

2. [**MDX 확장 문법**](./mdx.md)
   - React 컴포넌트, 탭, 알림 박스 등 더 풍부한 기능을 사용하는 법을 배웁니다.
   - 문서를 더 예쁘고 동적으로 만들고 싶다면 확인하세요.

3. [**지원 React 컴포넌트**](./react-components.md)
   - 현재 Bakerywave 문서에서 바로 사용할 수 있는 컴포넌트와 속성을 정리합니다.
   - 실제 구현 기준으로 확인하고 싶을 때 먼저 참고하세요.

4. [**API 링크 (백틱 문법)**](./api-links.md)
   - `Class.Part` 처럼 코드를 작성하면 자동으로 링크가 걸리는 마법 같은 기능을 소개합니다.
   - API 문서를 참조해야 할 때 필수적입니다.
