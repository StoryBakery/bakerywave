# Bakerywave란?

**Bakerywave**는 StoryBakery 팀이 개발한 **통합 문서화 도구**입니다.  
복잡한 웹 개발 지식 없이도, 누구나 **명령어 몇 줄**만으로 아름답고 기능적인 문서 사이트를 만들 수 있습니다.

## 주요 기능

### 1. 🚀 초고속 웹사이트 생성
Bakerywave는 세계적으로 인기 있는 문서 도구인 [Docusaurus](https://docusaurus.io/)를 기반으로 합니다.  
하지만 Docusaurus의 복잡한 설정은 저희가 미리 다 해두었습니다. 여러분은 **콘텐츠 작성**에만 집중하세요.

*   **반응형 디자인:** PC, 태블릿, 모바일 어디서든 완벽하게 보입니다.
*   **다크 모드:** 눈이 편안한 다크 모드를 기본으로 지원합니다.
*   **강력한 검색:** 문서 내용을 빠르게 찾을 수 있습니다.

### 2. 📝 Luau 코드 API 자동 문서화
Roblox 개발자라면 **Luau** 언어를 많이 사용하실 겁니다.  
코드 위에 주석만 잘 적어두면, Bakerywave가 이를 분석해서 **멋진 API 레퍼런스 문서**를 자동으로 만들어줍니다.

*   함수, 클래스, 타입 정보를 자동으로 추출합니다.
*   `@param`, `@return` 같은 태그를 인식해 표로 정리해줍니다.
*   네이티브 파서(`luau-docgen`)를 사용하여 매우 빠릅니다.

### 3. 🛠️ 간편한 통합 CLI
웹사이트 빌드, 문서 생성, 배포까지 모든 작업을 `bakerywave`라는 하나의 명령어로 처리할 수 있습니다.

## 누구를 위한 도구인가요?

*   **라이브러리 개발자:** 자신이 만든 모듈의 사용법을 남들에게 알리고 싶을 때
*   **팀 프로젝트:** 팀원들과 공유할 기술 문서를 만들 때
*   **튜토리얼 작성:** 게임 개발 강좌나 가이드를 작성할 때

이제 다음 단계로 넘어가서 실제로 Bakerywave를 설치하고 사용해봅시다!
