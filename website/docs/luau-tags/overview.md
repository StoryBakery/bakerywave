# Luau 주석 개요

Bakerywave는 Luau 소스 코드에 작성된 주석(Comment)을 읽어서 문서를 만듭니다.
이때 아무 주석이나 읽는 것은 아니고, **문서화 주석(Documentation Comment)**이라고 부르는 특별한 형태만 인식합니다.

## 주석 형태

반드시 `--[=[` 으로 시작하고 `]=]` 으로 끝나는 블록 주석을 사용해야 합니다.

```lua
--[=[
    여기에 설명을 적습니다.
    여러 줄을 적어도 됩니다.
    
    마크다운 문법도 사용할 수 있습니다.
    - **굵게**
    - *기울임*
]=]
local myValue = 10
```

일반적인 한 줄 주석(`--`)이나 짧은 블록 주석(`--[[ ... ]]`)은 **무시됩니다.**
문서로 만들고 싶은 부분만 `--[=[ ... ]=]`을 사용하세요.

## 작성 위치

주석은 설명하려는 **대상(함수, 변수, 타입 등)의 바로 윗줄**에 작성해야 합니다.

✅ **좋은 예:**
```lua
--[=[
    플레이어의 체력을 회복시킵니다.
]=]
function Player:Heal(amount: number)
end
```

❌ **나쁜 예 (떨어져 있음):**
```lua
--[=[
    플레이어의 체력을 회복시킵니다.
]=]

-- 다른 주석
function Player:Heal(amount: number) -- 인식을 못 할 수 있습니다.
end
```
